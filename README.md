## Loops in Python
* Introduction

Loops are used to execute a block of code repeatedly until a specified condition is met. They help reduce code duplication and make programs more efficient.

Python provides two main types of loops:

* for loop – Used to iterate over a sequence (list, tuple, string, dictionary, set, etc.).
* while loop – Used to repeat a block of code as long as a condition is True.

🔹 For Loop

A for loop is used to iterate over a sequence or collection of items.

Syntax :
for variable in sequence:
    # Code to execute

🔹 While Loop

A while loop executes a block of code as long as the given condition is True.

Syntax :
while condition:
    # Code to execute
